#  AI Agent for study



一个用于学习了解AI代理系统的项目，支持交互式对话和自动化任务执行。



## 项目概述



**AI Agent** 是一个模块化、可扩展的智能代理系统，旨在帮助开发者执行文件操作、代码分析、项目管理和自动化任务。系统采用清晰的架构设计，支持多种运行模式，具有良好的用户体验和扩展性。



## 核心特性



- **多种运行模式**: 支持单次任务执行、交互式对话和会话管理

- **模块化工具系统**: 易于扩展的工具集，支持文件操作、代码分析等功能

- **会话记忆**: 支持对话历史的保存、加载和总结

- **命令行界面**: 直观的命令行参数，支持详细输出和颜色控制

- **错误处理**: 完善的异常处理和用户中断支持

## 安装与运行



### 环境要求

- Python 3.6+

- 依赖包（见 requirements.txt）



### 安装步骤

1. 克隆项目或下载源代码

2. 安装依赖：

   ```bash

   pip install -r requirements.txt

   ```

3. 运行程序：

   ```bash

   python main.py --help

   ```



## 使用方式



### 单次任务模式

执行单个任务并退出：

```bash

python main.py "创建一个Python文件"

```



### 交互模式

进入交互式对话环境：

```bash

python main.py --interactive

```



### 会话管理

使用会话文件保存/加载对话历史：

```bash

# 使用会话文件

python main.py --session my_session



# 清除会话历史

python main.py --clear-session

```



### 工具管理

列出所有可用工具：

```bash

python main.py --list-tools

```



### 输出控制

```bash

# 显示详细输出

python main.py --verbose



# 禁用颜色输出

python main.py --no-color

```



## 命令行参数



| 参数 | 缩写 | 描述 |

|------|------|------|

| `task` | - | 要执行的任务描述（单次任务模式） |

| `--interactive` | `-i` | 进入交互模式 |

| `--session` | `-s` | 会话文件路径 |

| `--clear-session` | - | 清除会话历史 |

| `--list-tools` | - | 列出所有可用工具 |

| `--verbose` | `-v` | 显示详细输出 |

| `--no-color` | - | 禁用颜色输出 |



## 交互模式命令



在交互模式下，可以使用以下命令：



- `exit` 或 `quit`: 退出程序

- `help`: 显示帮助信息

- `clear`: 清空当前会话历史

- `history`: 显示历史记录

- `summary`: 显示记忆总结

- 其他任何输入都将作为任务执行



## 技术架构



### 核心模块

1. **ChatAgent**: 聊天代理核心，负责任务解析和执行

2. **Memory**: 记忆管理，支持对话历史的持久化存储

3. **Tools**: 模块化工具系统，支持各种功能扩展



### 设计原则

- **模块化**: 各功能模块独立，易于维护和扩展

- **可配置**: 通过配置文件调整系统行为

- **可验证**: 每个操作都有明确的验证机制

- **用户友好**: 直观的命令行界面和交互体验



## 开发指南



### 添加新工具

1. 在 `tools/` 目录下创建新的工具模块

2. 实现工具功能并注册到工具系统

3. 更新工具描述文档



### 修改配置

1. 编辑 `config/` 目录下的配置文件

2. 参考现有配置格式



### 测试

运行测试确保功能正常：

```bash

# 运行基本功能测试

python -m pytest tests/

```

3. **Python版本**: 确保使用 Python 3.6+ 版本
